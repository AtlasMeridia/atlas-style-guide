# Aesthetic Direction

This directory contains subjective, impressionistic notes on visual direction for ATLAS Meridia. Unlike the structured design tokens in `/data/tokens.json`, these are mood-based guidelines that inform creative decisions.

## Structure

- `print/` - Aesthetic direction for print materials
  - `stationery.md` - Letterhead, envelopes, note cards

## Reference Images

Curated reference images live in `/public/aesthetic/` and are viewable via the app.
